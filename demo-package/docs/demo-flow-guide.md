# TarifAL Demo Akış Rehberi

## Demo Süresi

3-5 dakika.

## Demo Başlamadan Önce

- Uygulama linkini aç: `http://localhost:8081/`
- Demo kullanıcıyla giriş yap.
- Ana sayfada **Yatırımcı Demosunu Başlat** kartını kullan.
- Demo siparişin gerçek sipariş olmadığını konuşmanın başında belirt.

## Demo Sırasında Dikkat Edilecekler

- Gerçek ödeme yapılmadığını açıkça söyle.
- Finansal projeksiyonların tahmini olduğunu belirt.
- Gelir modelini "potansiyel gelir modeli" olarak anlat.
- Ana hikayeyi kısa tut: **malzemeden tarife, tariften sepete, sepetten sofraya.**
